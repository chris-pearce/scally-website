Scally is a Sass-based, BEM, OOCSS, responsive ready, CSS framework that provides you with a solid foundation for building reusable UI's quickly. It is a framework focused on scalability and reuse, there when you need it and getting out of your way when you don't.

Scally is designed to be configurable, only requiring you to bring in the parts you are using, keeping your CSS architecture light weight and scalable. It is unopinionated about design giving you more flexibility than your typical UI toolkit.

Install via Bower

bower install scally

Building UI is easy with Scally

The Scally framework helps you construct UI really easily by providing you with a suite of powerful utilities and layout modules.

These utilities and layout modules can form a wide variety of UI patterns from simple principles meaning as CSS authors you don't have to keep writing the same styles over and over again, instead you can abstract those common styles into nice reusable chunks of code.

Some examples

Grid

A powerful fluid and nestable mobile first grid system

View HTML 
<div class="l-grid">
  <div class="l-grid__item  u-one-third-from-lap">
    [content]
  </div>
  <div class="l-grid__item  u-one-third-from-lap">
    [content]
  </div>
  <div class="l-grid__item  u-one-third-from-lap">
    [content]
  </div>
</div>
<div class="l-grid">
  <div class="l-grid__item  u-one-half-from-lap">
    [content]
  </div>
  <div class="l-grid__item  u-one-half-from-lap">
    [content]
  </div>
</div>
View Sass 
.l-grid {
  @include to-rem(margin-left, -$l-grid-gutter-base);
  display: block;

  &:after {
    content: "";
    display: table;
    clear: both;
  }
}

  .l-grid__item {
    float: left;
    @include to-rem(padding-left, $l-grid-gutter-base);
    width: 100%;

    @if $friendly-box-model == false {

      box-sizing: border-box;

    }
  }

Side-by-Side

Place any two elements side-by-side

View HTML 
<div class="u-side-by-side">
  <div class="u-side-by-side__left">
    [content]
  </div>
  <div class="u-side-by-side__right">
    [content]
  </div>
</div>
View Sass 
.u-side-by-side {
  @extend %u-clear-fix;
  display: block;
}

  .u-side-by-side__left {
    float: left;
    @include to-rem(margin-right, $u-side-by-side-gutter-base);
  }

  .u-side-by-side__right {@extend %u-nfbc;}

List Inline

Render a list horizontally

View HTML 
<ul class="u-list-inline">
  <li>[content]</li>
  <li>[content]</li>
  <li>[content]</li>
</ul>
View Sass 
%u-list-inline,
.u-list-inline {
  > li {display: inline-block;}
}
