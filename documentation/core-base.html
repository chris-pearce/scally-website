Links

Base link styles. At the bare minimum a colour is applied for both default and hover states. There is the option to remove the underline for the default state and apply it on the hover state instead, and an option to apply a bottom border as the underline instead of the browser default text-decoration: underline. You can also choose to apply a transition to link colour and its underline—if the bottom border option is selected—when hovering the link.
